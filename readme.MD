## fungsi sederhana golang

fungsi sederhana yang memanfaatkan slice partitioning

1. menyisipkan nilai berdasarkan index ke-n
2. mengcopy nilai array ke dalam array baru
3. membagi array lama ke dalam 2 array
4. gabungkan nilai index ke-n ke dalam array pertama
5. gabungkan array baru dengan array lama yang sudah ditambahkan nilai index ke-n
